# Midnight on Main

Midnight on Main is a first-person stealth puzzle escape game developed in **Godot**.  
The player explores a university engineering building after hours, collects missing homework assignments by solving puzzles, and avoids a roaming enemy known as the Homework Enforcer.

## How to Play

- Move using **W / A / S / D**
- Look around with the **mouse**
- Jump with **Space**
- Sprint by holding **Shift**
- Interact with objects by holding **E**
- Pause the game with **Esc**

---

## Objective

- Collect **5 homework books** by solving short problems
- Avoid the Homework Enforcer
- Escape the building to win
- Getting caught results in a loss

---

## Save System

- The game automatically saves after each book is collected
- Loading returns the player to the building with only the remaining books present

---

## Tools Used

- **Game Engine:** Godot
- **3D Modeling & Animation:** Blender
- **Audio:** Original music and sound effects created by the team

---

## Running the Game

### From Executable
- Download the Windows build
- Run `MidnightOnMain.exe`

### From Source
- Open the project in **Godot**
- Run the project from the editor
